# 已知限制和风险

## API 限制

### 1. 单文件大小限制

- **当前**：≤ 4GB（单 block 上传）
- **原因**：百度网盘新版 API 未实现真正的分片上传
- **计划**：v1.4 中实现分片上传，支持 ∞
- **用户规避**：拆分成多个小文件或压缩后上传

### 2. 批量上传性能

- **现象**：>1000 个小文件上传较慢
- **原因**：单文件 API 调用，无并发
- **建议**：先压缩为 zip 文件再上传
- **计划**：v1.5 中支持并发上传

### 3. 频繁分享触发风控

- **现象**：`errno=8001`（分享过于频繁）
- **解决**：24 小时内不要频繁分享同一路径
- **备注**：这是百度网盘的服务端限制

### 4. Cookie 有效期

- **BDUSS 有效期**：通常 1-3 个月
- **失效信号**：`errno=110` 或 `errno=-1`
- **解决**：重新从浏览器 F12 抓取
- **建议**：定期更新凭证

### 5. 并发限制

- **单账号限制**：百度网盘限制单账号的并发上传数
- **表现**：超过限制后会返回 `errno=500`
- **解决**：降低并发数或添加延迟

## 风险警告

### 🔴 安全风险

⚠️ **BDUSS 等同于账号密码**

- 泄漏后他人可完全控制你的网盘
- 不要在公开场合演示
- 不要硬编码到代码里
- 不要提交到 Git 版本控制
- 用完建议在设备管理中撤销

**最小权限原则**：
- 创建专用网盘账号用于自动化任务
- 定期更新凭证
- 监控网盘异常访问

### 🟡 API 稳定性风险

⚠️ **本工具依赖百度网盘 Web API 逆向**

- 百度可能在不通知的情况下改动 API
- API 升级可能导致脚本失效
- 建议定期运行 `quota` 命令检查连接状态
- 发现故障请在 GitHub Issues 中反馈

### 🟡 服务条款风险

⚠️ **使用本工具需遵守百度网盘服务条款**

- 不要用于违法用途
- 不要大规模采集公开分享内容
- 不要用于商业化转售
- 频繁操作可能被服务端限流

## 常见问题

### Q: 这个工具会被百度封禁吗？

A: 本工具遵守以下原则：
- 调用的是百度网盘公开的 Web API（与网页版一致）
- 不使用代理、不爬虫、不骚扰服务器
- 正常使用不会触发限制
- 频繁操作可能被限流（这是百度的正常防护）

### Q: 凭证会被记录到日志吗？

A: 不会。脚本的设计原则：
- BDUSS 仅从环境变量读取
- 日志中不会打印凭证
- 即使启用 `BAIDU_VERBOSE=1` 也不会泄露

### Q: 多个用户能共用一个 BDUSS 吗？

A: 理论上可以，但不建议：
- 无法区分谁操作了网盘
- 一个设备登出会影响其他设备
- 安全风险增加

建议：为每个用户/应用创建独立凭证

### Q: 可以用代理吗？

A: 可以，但需要手动修改脚本：
- 修改 `make_session()` 中的 `proxies` 参数
- 本项目不提供代理配置支持

## 迁移计划

如果百度网盘 API 失效，可考虑迁移到：

1. **官方 SDK**：百度网盘开放平台（需 OAuth）
2. **其他网盘**：OneDrive / Google Drive / 阿里云盘
3. **自建方案**：MinIO / Nextcloud 等开源方案

本项目会在 GitHub Issues 中及时通知任何 API 变更。
